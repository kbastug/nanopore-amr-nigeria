Antimicrobial Resistance Gene Detection
Antimicrobial resistance (AMR) gene detection was performed using publicly available web-based tools.

CARD (RGI)
Platform: Comprehensive Antimicrobial Resistance Database (CARD)
Analysis type: Whole-genome sequence (FASTA)
Organism selection: Species-specific when available
Threshold for reporting: 100% match
ResFinder
Platform: ResFinder via the Danish Technical University Center for Genomic Epidemiology
Analysis type: Whole-genome sequence (FASTA)
Threshold for reporting: 100% match
