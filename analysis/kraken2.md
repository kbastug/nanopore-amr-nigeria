Taxonomic Classification with Kraken2 (Galaxy)
Taxonomic classification of assembled contigs was performed using the Galaxy Bioinformatics platform (https://usegalaxy.org).

Tool
Kraken2 (v2.1.3)
Input
Assembled contigs (FASTA) generated by Flye
Database
Standard pre-built Kraken2 database (Galaxy)
Key Parameters
Confidence threshold: default
Taxonomic rank assignment: default
Output
Taxonomic classification reports
Taxon ID assignments used for species identification
